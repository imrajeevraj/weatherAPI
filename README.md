# Weather API Project

A simple web application that displays weather information using an external weather API.

## Project Structure

```
Weather API/
├── index.html      # Main HTML file
├── script.js       # JavaScript functionality
├── style.css       # Styling
└── README.md       # Project documentation
```

## Features

- Fetch current weather data
- Display weather information in a user-friendly interface
- Responsive design
- Real-time weather updates

## Getting Started

1. Open `index.html` in a web browser
2. Allow location access or enter a city name to get weather information
3. View the current weather conditions

## Files

- **index.html**: The main HTML page with the structure and layout
- **script.js**: Contains JavaScript code for API calls and DOM manipulation
- **style.css**: Stylesheet for the application's visual presentation

## How It Works

The application fetches weather data from an API and displays:
- Current temperature
- Weather conditions
- Humidity levels
- Wind speed
- And other weather details

## Browser Compatibility

Works on modern browsers that support:
- Fetch API
- Geolocation API (optional)
- ES6 JavaScript

## Notes

- Ensure you have an internet connection to fetch weather data
- API key may be required depending on the weather service used
- Check the JavaScript file for API configuration details
